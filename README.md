# 🎬 Movie Recommender System

A Machine Learning-powered Movie Recommendation System that recommends movies similar to a user's selected movie using **Content-Based Filtering** and **Cosine Similarity**. The application leverages movie metadata such as genres, keywords, cast, crew, and movie overviews to identify similar movies and provide highly relevant recommendations.

The project is deployed as an interactive web application using **Streamlit** and integrates with the **TMDB (The Movie Database) API** to fetch movie posters, metadata, and official trailers in real time.

# 📖 Project Overview

Finding movies to watch can be overwhelming due to the vast number of options available across streaming platforms. This project addresses that problem by recommending movies that share similar characteristics with a movie selected by the user.

Unlike popularity-based recommendation systems, this application focuses on the actual content of movies. By analyzing movie features and calculating similarity scores, it can suggest movies that are closely related in terms of themes, genres, cast members, directors, and story elements.

For example:

* If a user likes **Inception**, the system may recommend:

  * Interstellar
  * The Prestige
  * Shutter Island
  * The Dark Knight
  * Tenet

because these movies share similar genres, themes, directors, actors, or storytelling styles.

# 🧠 Machine Learning Approach

## Content-Based Filtering

The recommendation system uses Content-Based Filtering.

Instead of relying on user ratings, recommendations are generated by comparing movie content.

The system analyzes:

* Genre similarity
* Shared actors
* Shared directors
* Common keywords
* Similar plot descriptions

# 🔢 Feature Vectorization

The generated tags are converted into numerical vectors using Count Vectorizer.

Example:

Movie A:

```text
Action Adventure Sci-Fi
```

Movie B:

```text
Action Adventure Thriller
```

Vocabulary:

```text
Action Adventure Sci-Fi Thriller
```

Vector Representation:

Movie A:

```text
[1,1,1,0]
```

Movie B:

```text
[1,1,0,1]
```

---

# 📈 Similarity Calculation

The similarity between two movies is measured using Cosine Similarity.

Formula:

Cosine Similarity = (A · B) / (||A|| × ||B||)

---

## Example

Movie A (Inception)

```text
[1,1,1,1,0]
```

Movie B (Interstellar)

```text
[1,1,1,0,1]
```

### Dot Product

```text
(1×1)+(1×1)+(1×1)+(1×0)+(0×1)
= 3
```

### Magnitudes

Movie A:

```text
√(1²+1²+1²+1²+0²)
= 2
```

Movie B:

```text
√(1²+1²+1²+0²+1²)
= 2
```

### Final Similarity Score

```text
3 / (2×2)
= 0.75
```

Since the score is close to 1, the movies are considered highly similar.
